Installation:

Place the script folder in the resources directory of your FiveM server.

Ensure that 'start lone_polmegaphone' is added to your server.cfg file.

Adjust configuration settings in the 'config.lua' file to match your server's preferences.

Restart the server.

Usage:

Instruct law enforcement officers to use the designated button (default: lshift) the megaphone and adjust settings.

hold the button and speak to talk in megaphone
