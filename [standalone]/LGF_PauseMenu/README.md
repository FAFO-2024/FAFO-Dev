## Configuration Files:
- General Configuration: `LGF-PauseMenu\config.lua`
    - Set Server Name, Discord Invite, Language, Debug Mode
- Translations: `LGF-PauseMenu\config.json`
    - Follow the given structure, if not you <b>will</b> get errors
